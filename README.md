# Data Engineering Zoomcamp 2026

## ENV Setup

### Use `uv`

See more on https://github.com/astral-sh/uv

### Init `uv`

```bash
uv init --python=3.13
```

### Setup VENV and enable it

```
uv venv

source .venv/bin/activate
```

### Install third-party packages

All package use must be put on `./requirements.txt`

```
pip install -r requirements.txt

```
